# Deep Learning Basics 🚀

This README gives a simple introduction to key deep learning concepts.

---

## 1. What is Deep Learning?
Deep Learning is a part of Machine Learning that uses **neural networks with many layers** to learn patterns from data.

It is inspired by how the human brain works.

---

## 2. Where is it used?
- Image recognition (face unlock, Google Photos)
- Language translation (Google Translate)
- Chatbots (ChatGPT)
- Self-driving cars
- Recommendation systems (Netflix, YouTube)

---

## 3. Basic Building Block: Neuron 🧠

A neuron:
- Takes inputs
- Applies weights
- Adds bias
- Uses an activation function
- Produces output

---

## 4. Neural Network Structure

A neural network has:
- **Input layer** → takes data
- **Hidden layers** → processes data
- **Output layer** → gives result

Example:
Input → Hidden Layer(s) → Output


---

## 5. Activation Functions ⚡

They help decide if a neuron should "fire" or not.

Common ones:
- ReLU (Rectified Linear Unit)
- Sigmoid
- Softmax

---

## 6. Forward Propagation
Data moves from input → output through the network.

---

## 7. Loss Function 📉
It measures how wrong the model is.

Goal: **Reduce loss**

Examples:
- Mean Squared Error (MSE)
- Cross Entropy Loss

---

## 8. Backpropagation 🔁
It is the process of:
- Calculating error
- Sending it backward
- Updating weights

This is how the model learns.

---

## 9. Optimizer ⚙️
Helps improve learning by updating weights.

Common optimizers:
- Gradient Descent
- Adam

---

## 10. Training Process 🏋️
1. Give input data
2. Forward pass
3. Calculate loss
4. Backpropagation
5. Update weights
6. Repeat

---

## 11. Overfitting vs Underfitting

- **Overfitting** → model memorizes data (bad)
- **Underfitting** → model doesn't learn enough (bad)
- **Good model** → balanced learning

---

## 12. Summary ✨
Deep Learning = Neural Networks + Data + Training

It learns patterns automatically with large datasets and computing power.
