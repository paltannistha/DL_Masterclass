# 🧠 Day 3: Model Evaluation – Overfitting vs Underfitting

## 📌 Overview
This project demonstrates how to evaluate a simple neural network model using **train-test split** and understand two important problems in machine learning:

- Overfitting  
- Underfitting  

The goal is to learn how well a model generalizes to unseen data.

- Creating a synthetic dataset

- Training a neural network using `MLPClassifier`  
- Evaluating model performance using accuracy  
- Understanding overfitting and underfitting
