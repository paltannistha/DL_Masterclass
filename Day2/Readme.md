# 🧠 Simple Neural Network from Scratch

This is a beginner-friendly implementation of a neural network using pure Python.

It demonstrates how a model can learn a simple pattern:
> **y = 2x**

---

## 🚀 What this project does

- Takes input values (X)
- Predicts output using weight and bias
- Calculates error (loss)
- Updates itself to improve predictions
- Repeats the process until it learns the correct pattern

---

## 🔁 How it works

The model follows a simple loop:

1. Make a prediction  
2. Calculate error  
3. Adjust weight and bias  
4. Repeat  

Over time, the error decreases and predictions improve.

---

## 🧪 Example

**Input:**
1, 2, 3, 4, 5

**Expected Output:**
2, 4, 6, 8, 10


## 💻 How to run

```bash
python first_neural_network.py
